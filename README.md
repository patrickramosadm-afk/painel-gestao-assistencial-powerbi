# **Painel de Gestão Assistencial**

## **Objetivo**

Desenvolver um dashboard em **Power BI** para acompanhar indicadores assistenciais de uma unidade de saúde, consolidando informações que apoiam a tomada de decisão e o monitoramento de metas.

## **Funcionalidades**

- Indicadores de usuários ativos
- Busca ativa
- Produção mensal
- Altas
- Monitoramento de metas assistenciais
- Evolução anual da produção

## **Ferramentas**

- Power BI
- Excel
- DAX
